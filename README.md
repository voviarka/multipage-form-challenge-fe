# Multipage form challenge

NodeJS v 23.0

1) To run the project:

`pnpm install`
`pnpm dev` - the server will start on port 5173.


2) To run the e2e tests:

`pnpm dev` - run server
`pnpm playwright test --ui` - the command will run playwright in the visual mode
